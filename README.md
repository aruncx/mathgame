Number Jungle — Addition & Subtraction Adventure

A friendly math practice game for Class 1 and 2 students. Students solve fill-in-the-blank equations (like 6 + ▢ = 8) by tapping the correct number from a set of choices. A monkey mascot hops along a vine to show progress — clapping and celebrating with confetti and cheerful sounds on correct answers, and gently wobbling with a buzz on wrong ones, so kids get instant, encouraging feedback either way.

Before starting, the teacher or student picks:

Operation — addition, subtraction, or both
Difficulty — the biggest number allowed in the answer (5, 10, 20, 50, or 100), so it scales from beginners to more advanced students
Number of questions — 5, 10, 15, or 20 per round

Students can move back and forth between questions with Next/Back buttons, and at the end they see a summary screen with their correct and wrong counts, plus options to play again or change the settings.

It's a single self-contained HTML file — no installation needed, just open it in a browser.
